# My Mobile App - Task List Prototype

A simple React Native mobile app prototype built with Expo.

## 🎯 What This App Does

A task list app demonstrating core mobile UI patterns:
- Add new tasks
- Mark tasks as complete/incomplete
- Delete tasks
- Scrollable list view
- Modern, clean UI design

## 🚀 Quick Start

### View in Browser (Easiest - Running Now!)
```bash
npm run web
```
Opens at http://localhost:8081

### View on Your Phone (Best for Mobile Testing)
1. Install **Expo Go** app on your phone
   - iOS: App Store
   - Android: Google Play Store
2. Run `npm start`
3. Scan the QR code with your phone

### View in iOS Simulator (Requires Xcode)
```bash
npm run ios
```
See `INSTALL_XCODE.md` for installation instructions.

### View in Android Emulator (Requires Android Studio)
```bash
npm run android
```
See `INSTALL_ANDROID_STUDIO.md` for installation instructions.

## 🛠️ Development

### Edit the App
Open `App.js` in your editor and make changes. The app will automatically reload.

### Available Scripts
- `npm start` - Start Expo dev server with QR code
- `npm run web` - Open in web browser
- `npm run ios` - Open in iOS simulator
- `npm run android` - Open in Android emulator
